## How to compile and run (C++ examples)

Most code files are single-file C++ programs. To compile and run a file using g++ (MinGW or WSL), use:

```powershell
# Compile
g++ -std=c++17 -O2 -Wall "path\to\file.cpp" -o "path\to\output.exe"

# Run
"path\to\output.exe"
```

If you're using Visual Studio Code, you can use the C/C++ extension or configure a tasks.json to build and run files easily. Some example files include their own comments about input format.

## How to use (quick)

1. Pick a folder/topic (e.g., `Sorting/`).
2. Open a file (e.g., `SelectionSorting.cpp`).

4. Compile and run with sample input (if any).
